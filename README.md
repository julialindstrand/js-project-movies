# Project info
Multi page application showing popular movies from TMDB API. Built using React Router, useState and useEffect. Movies listed on home page, re-routing to info page about each movie when clicked. Fully responsive from 320px to 1600px. 

## Tech
* React, React router
* Styled components
* TMDB API
